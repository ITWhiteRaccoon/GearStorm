Default Keys

Here is a list of all the current key configs and actions you can take:, you can change these keys from default to whatever you want in the main menu:
  1. In Game Menu : Escape
  2. Scoreboard : Tab
  3. Fire : Left Mouse Button
  4. Targeting : Right Mouse Button
  5. Next Weapon : Mouse Scroll Up
  6. Previous Weapon : Mouse Scroll Down
  7. Reload : R
  8. Run : Left Shift / Divide
  9. Jump : SpaceBar
  10. Toggle Chat : C
  11. Use : E / Middle Mouse Button
  12. Context Menu : F
  13. Toggle Inventory : Page Down
  14. Build Gun Roll : Z / Num Pad 1
  15. Build Gun Pitch : X / Num Pad 7
  16. Build Gun Yaw : Y / Num Pad 7
  17. Crouch : Right Ctrl
  18. SaveMenu : F7
  19. HandBrake : Add
  20. ToggleInventory : I/Page Down
  21. Toggle Gun Configuration : G/Num Pad 5
  22. Push To Talk : T
  23. Chat : C
  24. Toggle 3rd Person: Home
  25. MouseLook : Insert
  26. LightToggle : L
  27. Vehicle Grab : G
  28. Vehicle Deploy : H
  29. Toggle Cheat Menu : Backspace
  30. Toggle Map : M
  31. Reload : R
  32. Crouch/Prone : LeftControl
  33. Toggle HUD : V
  34. Auto Roll Stabilization : LeftControl
  35. ToggleAdminMenu : BackSpace
  36. GunRoll : Z
  37. GunPitch : X
  38. ThirdPersonToggle : LeftControl
  39. Toggles Fullscreen : Alt-Enter, F-11
