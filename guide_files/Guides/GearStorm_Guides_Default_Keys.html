Default Keys

Here is a list of all the current key configs and actions you can take:, you can change these keys from default to whatever you want in the main menu:

  • In Game Menu : Escape
  • Scoreboard : Tab
  • Fire : Left Mouse Button
  • Targeting : Right Mouse Button
  • Next Weapon : Mouse Scroll Up
  • Previous Weapon : Mouse Scroll Down
  • Reload : R
  • Run : Left Shift / Divide
  • Jump : SpaceBar
  • Toggle Chat : C
  • Use : E / Middle Mouse Button
  • Context Menu : F
  • Toggle Inventory : Page Down
  • Build Gun Roll : Z / Num Pad 1
  • Build Gun Pitch : X / Num Pad 7
  • Build Gun Yaw : Y / Num Pad 7
  • Crouch : Right Ctrl
  • SaveMenu : F7
  • HandBrake : Add
  • ToggleInventory : I/Page Down
  • Toggle Gun Configuration : G/Num Pad 5
  • Push To Talk : T
  • Chat : C
  • Toggle 3rd Person: Home
  • MouseLook : Insert
  • LightToggle : L
  • Vehicle Grab : G
  • Vehicle Deploy : H
  • Toggle Cheat Menu : Backspace
  • Toggle Map : M
  • Reload : R
  • Crouch/Prone : LeftControl
  • Toggle HUD : V
  • Auto Roll Stabilization : LeftControl
  • ToggleAdminMenu : BackSpace
  • GunRoll : Z
  • GunPitch : X
  • ThirdPersonToggle : LeftControl
  • Toggles Fullscreen : Alt-Enter, F-11
